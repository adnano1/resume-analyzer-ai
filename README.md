# AI Resume Analyzer

An AI-powered web application that analyzes resumes, extracts skills using NLP, and compares them with job descriptions to generate a match score.

---

## 🚀 Features

- Upload resume (PDF)
- Extract skills automatically
- Compare resume with job description
- Generate job match score
- Display matched skills
- Clean web interface using Flask + Bootstrap

---

## 🧠 How It Works

1. User uploads a resume.
2. The system extracts text from the resume.
3. NLP is used to detect skills.
4. Skills are compared with the job description.
5. A match score is generated.

## ⚙️ Installation

Clone the repository


git clone https://github.com/adnano1/resume-analyzer-ai.git


Move into project folder


cd resume-analyzer-ai


Create virtual environment


python -m venv venv


Activate environment

Windows:


venv\Scripts\activate


Install dependencies


pip install -r requirements.txt


Run the application


python app.py


Open in browser


http://127.0.0.1:5000
